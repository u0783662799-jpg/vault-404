# Flossa Mobile App

Minimalny szkielet React + Vite + TypeScript + Tailwind przygotowany mobile first.

## Struktura

- `src/app` - konfiguracja aplikacji i root React.
- `src/components` - współdzielone komponenty UI.
- `src/features` - moduły produktowe i przyszłe ekrany.
- `src/hooks` - współdzielone hooki.
- `src/lib` - narzędzia, adaptery i konfiguracje.
- `src/styles` - globalne style i tokeny Tailwind.
- `src/types` - współdzielone typy TypeScript.

## Komendy

```bash
npm install
npm run dev
npm run build
npm run lint
```
